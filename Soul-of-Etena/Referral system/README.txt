Реферальная система для Telegram-бота
Этот скрипт представляет собой простую реферальную систему для Telegram-бота.
Он позволяет пользователям регистрироваться и получать код приглашения, который можно использовать для приглашения других пользователей.

Установка
Создайте новый файл referral_system.py и скопируйте в него содержимое этого скрипта.
Замените YOUR_TELEGRAM_BOT_TOKEN на токен вашего Telegram-бота.
Установите необходимые библиотеки, выполнив команду pip install python-telegram-bot sqlite3.
Создайте базу данных referrals.db и таблицу referrals с полями id, user_id и referrer_id.
Запустите скрипт, выполнив команду python referral_system.py.
Использование
Отправьте команду /start в Telegram-боту, чтобы запустить реферальную систему.
Реферальная система отправит сообщение с просьбой ввести код приглашения.
Если пользователь вводит код приглашения, реферальная система регистрирует его и отправляет сообщение с кодом приглашения.
Пользователь может использовать код приглашения для приглашения других пользователей.
Отправьте команду /check_referral в Telegram-боту, чтобы просмотреть список рефералов.
